1  Register Your Sender Address

Register MC Store's pickup address with Shipbubble. You'll get a code — copy it to Vercel Environment Variables as SHIPBUBBLE_SENDER_CODE.

2  Add Code to Vercel
  1. Copy the address code from Step 1 above
  2. Open Vercel → Your admin project → Settings
  3. Click Environment Variables → Add New
  4. Name: SHIPBUBBLE_SENDER_CODE
  5. Value: paste the code
  6. Click Save then Redeploy the project
